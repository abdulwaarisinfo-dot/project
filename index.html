How it works

One brain, every install

PHPTRAVELS is self-hosted. The agent isn't — and that's what makes it licensable.

1

The agent runs as a hosted service

One system, maintained centrally. Agencies never install, update or manage a model.

2

Each install talks to it over the existing API

The agent reads live inventory and writes bookings through the same endpoints PHPTRAVELS already exposes. No change to the core product.

3

Every agency configures its own

Their policies, their fare rules, their markup, their tone, their brand. Nothing is shared between agencies.

4

It hands over when it should

Low confidence, group travel, anything outside policy — routed to a human with the full conversation attached.

5

PHPTRAVELS sells it as a module

Listed alongside flights, hotels and visas. Billed monthly, or per completed booking.
